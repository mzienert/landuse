Chapter 1 GENERAL PROVISIONS
Sec. 1-10. Amendments to Code.
All ordinances or resolutions passed subsequent to this
Code
of La Plata
County
, Colorado, which amend, repeal or in any way affect this Code,
may
be numbered in accordance with the numbering system of this Code and printed for inclusion therein, or in the case of repealed chapters, sections and subsections or any part thereof, by subsequent ordinances or resolutions, such repealed portions may be excluded from the Code by omission from reprinted pages affected thereby, and the subsequent ordinances or resolutions as numbered and printed or omitted, in the case of repeal,
shall
be prima facie evidence of such subsequent ordinances or resolutions until such time that this Code and subsequent ordinances or resolutions numbered or omitted are readopted as a new code.
Amendments to any of the provisions of this Code should be made by amending such provisions by specific reference to the section of this Code in substantially the following language: "That section ____________ of the Code of La Plata County, Colorado, is hereby amended to read as follows:….(set out new provisions in full)."
If a new section not heretofore existing in the Code is to be added, the following language may be used: "That the Code of La Plata County, Colorado, is hereby amended by adding a section (or article, chapter or other designation as the case may be), to be numbered ____________, which reads as follows:….(set out new provisions in full)."
All sections, articles, chapters or provisions of this Code desired to be repealed should be specifically repealed by section number or chapter number, as the case may be.
(Res. No. 1997-61, 10-14-1997)
Effective on: 1/1/1901
